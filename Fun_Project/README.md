# Hangman Game

## Description
A simple, interactive Handman game, with various difficulty levels, hint system, visual representation  of the hangman and multiple categories.

## Dataset
- Source: Text Files
- Files:
  - programming_words.txt
  - sports.txt
  - countries.txt
- Description: Words are loaded dynamically from these files and categorized accordingly.

## Features
- Difficulty levels (Easy, Medium, Hard)
- Multiple categories (Programming, Sports, Countries)
- Hint system (limited and strategic)
- Option to guess the full word (if known)
- ASCII-based visual hangman
- Exit option at any point in the game

## Steps Performed
1. Loaded lists of words from different categories (Programming, Sports, Countries), from external text files
2. Categorized words into themes in the code
3. Implemented difficulty filtering based on word length
4. Built interactive game loop
5. Added hint system with limits
6. Designed a simple interface, with user input and an option to end the game at any point.


## Results
- Interactive and engaging gameplay
- Dynamic word selection
- Balanced difficulty system

## Tools Used
- Python
- Random module

## Conclusion
This project shows the process of loading files, designing a simple user interaction interface, and implementing structured programming.
